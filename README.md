# SimplifyFractions

This is a gem to simplify fractions

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'simplify_fractions'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install simplify_fractions
